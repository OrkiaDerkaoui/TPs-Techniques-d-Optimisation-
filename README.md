# TPs-Techniques-d-Optimisation-
Cours, TD et TP de Techniques d’Optimisation : programmation linéaire, méthodes exactes, heuristiques, meta-heuristiques, simplexe, transport, affectation et optimisation combinatoire.
# Techniques d’Optimisation

## Présentation

Bienvenue dans le dépôt du module **Techniques d’Optimisation**.

Ce dépôt contient :

- les supports de cours ;
- les travaux dirigés (TD) ;
- les travaux pratiques (TP) ;
- les projets ;
- les examens ;
- des exemples d’implémentation en Python.

Les TP sont réalisés principalement avec **Python** afin d’appliquer les méthodes d’optimisation étudiées en cours.

---

# Objectifs du module

Ce module a pour objectif de permettre aux étudiants de :

- comprendre les problèmes d’optimisation ;
- modéliser un problème mathématiquement ;
- appliquer les méthodes exactes ;
- résoudre des problèmes de programmation linéaire ;
- utiliser Python pour implémenter les algorithmes.
